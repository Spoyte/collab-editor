# Collab Editor

Real-time collaborative coding environment

## Features

- JavaScript/TypeScript support
- CLI interface
- Easy to use
- Lightweight and fast

## Installation

```bash
npm install -g collab-editor
```

## Usage

```bash
# Run the script
python server.js

# Or make executable and run
chmod +x server.js
./server.js
```



## Configuration

Configuration can be done via JSON/YAML files in the project root.
